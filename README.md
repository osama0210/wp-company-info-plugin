# WP Company Info - Plugin & Theme

## Description
This repository contains a custom WordPress plugin and theme for managing and displaying company information.

## Main Features:

### Plugin
- A box on the Dashboard with company info.
- A settings page to fill in: Name, Address, Phone, and Email.
- A media page to upload images using the WordPress Media Library.
- An image slider for the front of the website.
- Fully translated into Dutch and English.

### Theme
- Header with primary navigation and footer with footer navigation.
- Custom Post Type for team members with metaboxes (function, email, LinkedIn).
- Homepage slider using Swiper.js.
- Archive and single pages for team members.
- Custom 404 page.
- Fully translated into Dutch and English.

## How to use it

### Plugin
1. Put the plugin folder in your plugins directory.
2. Activate the plugin.
3. Go to "Company Info" in the menu to fill in your data.
4. Go to "Company Info > Media" to pick images for the slider.

### Theme
1. Put the theme folder in your themes directory.
2. Activate the theme in the WordPress dashboard.
3. Go to "Appearance → Menus" to set up your navigation and footer menus.
4. Go to "Team Members → Add New" to add team members.

### Shortcodes
You can use these codes in any page or post:
- `[ci_name]` : Shows company name.
- `[ci_contact_name]` : Shows contact person.
- `[ci_phone]` : Shows phone number.
- `[ci_email]` : Shows email.
- `[ci_address]` : Shows address.
- `[ci_slider]` : Shows the image slider.
- `[team_slider]` : Shows the team members slider.
